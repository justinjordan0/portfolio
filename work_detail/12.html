About Project

Face Mask Detection application take inputs from camera and make detection's about presence of face mask or not. Face Mask detection application take video input from both front and back cameras. Face Mask detection checks either user is wearing face mask, if yes, then it will display result as face mask detected. If user is not wearing face mask then face mask detection show result as face mask not detected and make red square on it. Face mask detection work on individual as well as on group of users and detect face mask is wearied or not. Face mask detection and recognition of face mask accuracy is too good.

Key

Android-Kotlin, TensorFlow, NDK
